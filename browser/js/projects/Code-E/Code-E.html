Description:

Code-E is a chrome extension that uses language recoginition in order to add voice functionality to chrome.

Code-E was originally concieved of as an Amazon Echo clone and later refined to function mainly as a developer tool for quick lookup of docs. Basic browser functionality such as the opening and closing of tabs and searching is also included.

Code-E can even tell you jokes and the weather! Check out the video!
