Description:

Angular impelementation of a card game called 'Set'. The object of the game is to find sets of 3 cards before time runs out and the computer takes a set. For a set to be valid all 3 of their respective backgrounds, colors, numbers, or shapes must be either all the same or all different.

Example Valid Sets: Solid One Red Diamond, Striped Two Purple Squiggle, Faded Three Green Lemon
Solid One Green Squiggle, Solid One Purple Diamond, Solid One Red Lemon

Example Invalid Set: Faded Two Green Diamond, Solid One Green Diamond, Striped Three Red Diamond
* This set is invalid because the 3rd card does not match in color.
